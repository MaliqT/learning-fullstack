    Using the mark element, we can highlight text.

    Semantics are a way to organize not only the web page but the code as well. Rather than using div and span that have no semantic meaning, we can specify parts of our code for assisted tools to recognize those parts. So far for semantics, I've used header, main, article, section, aside, details, summary, mark, and time.
